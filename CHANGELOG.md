# Changelog

## [1.0.0] - 2026-03-26

### Added
- Initial release
- Kanban board with drag-and-drop
- Task list view with advanced filters
- Search functionality
- Bulk actions (move, assign, delete)
- Comments with pinning
- Task history/audit trail
- Email notifications
- File attachments (PDF, images, Office documents)
- Subtasks with progress tracking
- Task dependencies (blocked by / blocks)
- Recurring tasks (daily, weekly, monthly, yearly)
- Dashboard widgets (My tasks, Overdue, Total, By status)
- Time tracking integration
- Full French and English translations
